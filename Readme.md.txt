#Fresh  Tomatoes Movie Trailers

This website show the trailer of the movie.

##Prerequistics
*Install Python [here](https://www.python.org/)


##Installation

There are three python file 
*media.py
*fresh_tomatoes.py
*entertainment.py
Open the**entertainment.py** file.

##Description
*In **media.py** there is a class name **Movie** which takes the date of the movie like movie-name,story-line etc.

*In **fresh_tomatoes.py** there is no class but this python file is the base of the project. It generates the dynammically webpages.

*In **entertainment.py** it uses the module media.py and fresh_tomatoes.py. using the data it generates the website.
